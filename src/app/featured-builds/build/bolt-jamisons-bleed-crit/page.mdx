> _Summary from the video_:
>
> This build is a crit build that focuses on bleed damage. While it excels at burning down bosses quickly, it is not as good at clearing mobs since the Corrupted Merciless has a hard time under pressure. But if the boss has a weakspot, this build will melt them.
